# CS2-Templates
CS2 templates: basic files to start a new mod

Start template layout

`YourModTemplate/
├─ .editorconfig
├─ .gitignore
├─ LICENSE
├─ README.md
└─ src/
   ├─ YourMod/                ← project folder (rename for your mod)
   │  ├─ YourMod.csproj
   │  ├─ Mod.cs
   │  ├─ Setting.cs
   │  ├─ LocaleEN.cs
   │  └─ Systems/
   │     └─ ExampleSystem.cs
   └─ YourMod.sln`
